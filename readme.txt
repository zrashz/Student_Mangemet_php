# 📚 Student Management System (PHP & MySQL)

## 🌟 Project Overview
The **Student Management System** is a web-based application that helps manage student records efficiently. It allows administrators to add, update, delete, and view student details. The system is built using **PHP** and **MySQL**, making it easy to deploy on local servers like XAMPP, WAMP, or LAMP.

## 🚀 How to Run the Project

### 1️⃣ Download & Extract
- 📥 **Download** the project zip file.
- 📂 **Extract** the file and locate the `studentrecordms` folder.

### 2️⃣ Move to the Server Directory
- Paste the `studentrecordms` folder inside your server root:
  - 🛠 **XAMPP:** `xampp/htdocs`
  - 🛠 **WAMP:** `wamp/www`
  - 🛠 **LAMP:** `var/www/html`

### 3️⃣ Set Up the Database
- Open **PHPMyAdmin** in your browser: [`http://localhost/phpmyadmin`](http://localhost/phpmyadmin)
- ➕ **Create a new database** named `studentrecorddb`.
- 📤 **Import** the `studentrecorddb.sql` file (located inside the SQL folder in the zip package).

### 4️⃣ Run the Project
- 🌐 Open your browser and enter: [`http://localhost/studentrecordms`](http://localhost/studentrecordms)

## 🛠 Prerequisites
Before running the project, make sure you have:
- ✅ XAMPP, WAMP, or LAMP installed.
- ✅ PHP and MySQL configured.
- ✅ A working browser like Chrome, Firefox, or Edge.

## ❓ Troubleshooting
💡 **Database connection error?** Double-check that the database name matches `studentrecorddb` and the SQL file is imported correctly.  
💡 **Can't access the project?** Ensure that XAMPP/WAMP/LAMP is running and Apache & MySQL services are started.
